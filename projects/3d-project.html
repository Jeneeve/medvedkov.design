Blender + Cinematic Animation

Overview

This project was created as part of a school assignment focused on 3D modeling, scene composition, lighting, and animation in Blender.

The goal was to build a stylized cinematic environment and explore motion, atmosphere, and visual storytelling through camera animation and rendering techniques.

The entire workflow included modeling, texturing, material setup, lighting, animation, rendering, and post-production optimization for web presentation.

Structure

3D Production

  • Scene modeling and object creation
  • Material and texture setup
  • Stylized environment design
  • Lighting and cinematic composition

Animation

  • Camera movement and transitions
  • Object animation and timing
  • Loop optimization for web
  • Visual storytelling through motion

Tools Used

  • Blender
  • Rendering & compositing
  • Video export optimization
  • Web integration
