DOMUS

OVERVIEW

Domus is a concept-driven digital product that explores how people interact with urban space through technology.

The project combines web and mobile interfaces, a custom 3D map experience built on Mapbox, and a cohesive visual identity to create an immersive and intuitive spatial service.

The goal of Domus was to rethink how location-based information can feel tactile, visual, and human-centered — rather than purely functional.

STRUCTURE

  • Web version (responsive)
  • Mobile application design
  • Custom 3D map concept using Mapbox
  • Logo design
  • Visual identity system
  • Posters and graphic materials
